# Contribution guidelines

Thank you for considering to contribute!

If you encounter an issue or have a feature request, you may [open an issue](https://github.com/urholaukkarinen/egui-toast/issues/new/choose) for it. You may also open a pull request for your contribution, but for larger changes, consider opening an issue first.

Keep pull requests focused. Avoid including multiple unrelated changes in the same pull request.
